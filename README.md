# AI Mentor

AI Mentor is a web-based personal growth and learning assistant that guides users through intelligent chat, tracks learning progress, and builds positive habits using AI.

## Core Features
- AI-powered mentorship chat
- Progress tracking (streaks, goals, topics)
- Habit building & reinforcement
- Personalized suggestions

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- AI: OpenAI API
